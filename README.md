# Inventario Frontend

## Descripción
Este repositorio contiene el frontend del proyecto de gestión de inventarios.  
Se encarga de la interfaz de usuario, la interacción con la API y la experiencia visual.

## Tecnologías
- HTML / CSS / JavaScript
- Frameworks: React / Vue / Angular (indica el que uses)
- Axios o Fetch para llamadas a la API

## Instalación
1. Clonar el repositorio:
```bash
git clone <URL del frontend>
Entrar en la carpeta del proyecto:

cd inventario-frontend


Instalar dependencias:

npm install

Uso

Iniciar el servidor de desarrollo:

npm start


Acceder a la aplicación en el navegador en:

http://localhost:3000

Estructura de Carpetas
frontend/
│
├── public/          # Archivos públicos (index.html, favicon, etc.)
├── src/             # Archivos fuente de la aplicación
│   ├── components/  # Componentes reutilizables
│   ├── pages/       # Páginas principales
│   ├── services/    # Funciones de conexión con la API
│   └── App.js       # Punto de entrada de la app
├── package.json     # Dependencias y scripts del proyecto
└── .gitignore       # Ignora node_modules y archivos sensibles

Notas

node_modules no se suben al repositorio.

Cada proyecto (frontend y backend) tiene su propio repositorio.
